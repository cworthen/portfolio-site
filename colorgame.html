RGB Color Game

Summary:

The Great RGB Color Game was created as part of the Web Developer Bootcamp by Colt Steele. I continued with the Color game and added several tweaks not included in the course. Aiming to dig deeper into Javascript but also get hands-on experience, this project has really helped further my understanding of DOM Manipulation such as selecting DOM elements, creating event listeners and dynamically changing CSS styles to elements.

Challenges:

A challenge I had during this project was that my color selection wasn't matching the random provided RGB color at the top of the page, causing correct choices to show as "Wrong". The solution was to put spaces after the commas in the RGB color value. This made sure that my return value for the randomColor function "rgb(" + r + ", " + g + ", " + b + ")"; matches what's displayed on top the screen "rgb(255, 105, 98)" Without the spaces after the commas rgb(255,105,98) the selected square would never match the randomly provided RGB color.

What I learned:

I learned how to pair logic with advanced DOM Manipulation. This has helpled to move forward and add several features to the game, including sounds effects and a toggle button to turn sound on/off, a "help" button for instructions on how to play the game and added level featuring 9 squares.

Technology used:

HTML5, CSS3, Javascript
