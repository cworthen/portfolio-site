Take App

Summary:

For this project, I converted the "Take" psd template into a responisve landing page. I used Bootstrap 4, HTML5 and CSS3 along with Javascript to make the page more interactive. "Take" is a fictional mobile app designed for taking photos and photo efects. You can see the original psd here, created by Blazrobar.

Challenges:

Trying to get the image and multicolored box grid responsive was a challenge. I had to figure the right combination of column widths to get the box grid to display just like the psd but also collaspe in a way that doesn't break the design.

What I learned:

From this project, I learned how to take a design and code into reality. This not only strengthed my understanding of bootstrap but also stretched my ability to pay close attention to detail.

Technology used:

HTML5, CSS3, Bootstrap 4, Media Queries, Javascript
