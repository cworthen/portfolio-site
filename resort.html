Pacific Trails Resort

Summary:

During my enrollment in the Web Developement program at Delaware County Community college, I completed several courses including HTML5 and CSS3, PHP, Javascript/Jquery and mobile web development. I created this website from scratch as part of the HTML/CSS course. The goal was to learn how to build a responsive website using HTML5, CSS3 and media queries. I was given guidelines on what to add for the website and how to make it look. This process taught me the basics of HTML and CSS such as building a navigation bar, creating a table with cells and borders, adding an image, creating a footer and linking to other pages. I also learned the different types of cascading styles sheets such as (Internal, Embed, External and Inline) and how to use them.

Challenges:

In the beginning, I struggled on understanding the difference between a div id and a class. I also had difficulty using media queries to scale down different elements and figure out the proper length values.

What I learned:

In class, the teacher introduced W3C Markup Validation Service, which I learned is a powerful tool. By fixing the errors, I became more aware of common mistakes made during coding and how to avoid them. I also realized the power of Google. Researching problems I'm having during coding has helped me so much during this process. It led me to great online resources such as Stack Overflow, CSS Tricks, Mozilla Developer Network, Dev Tips and many others.

Technology used:

HTML5, CSS3, Media Queries, Google Developer Tools
